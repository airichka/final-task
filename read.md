# Финальное задание

Программа, которая из имеющихся массива строк формирует новый, длинна которых меньше или равно 3.

Массив можно ввести с клавиатуры.

### Что умеет:
 - Ввод нескольких слов через запитую;
 - Проверка на валидное значения;
